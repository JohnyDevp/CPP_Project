APP USAGE

In main window you can either load new class diagram from json file (example in /data folder) or
create new one.

You can add new class by clicking on ADD CLASS button. Create relations by double clicking one the edge of one class and
then double clicking on the edge of another class. Edit by double clicking on what you want to edit.

Close all diagrams by clicking on CLOSE button.
